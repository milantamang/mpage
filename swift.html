Swift

Swift is a programming language.

  • print() - display text.
  • Simicolon (;) - not necessary at the end of every statement.
  • Constant (let) and Variable (var) - Constant cannot change and variable can change.
  • Explicit Type (:) - specify the type by writing it after the variable, separated by a colon.
  • Convert (+) - using plus operator.
  • Backslash(\) - Write the value in parentheses, and write a backslash (\) before the parentheses.
  • Three double quotation marks (""") - use for strings that take up multiple lines. Indentation at the start of each quoted line is removed.
  • Array ([]), - Create arrays and dictionaries using brackets and access their elements by writing the index or key in brackets. A comma is allowed after the last element.
  • Empty Array or dictionary [].
